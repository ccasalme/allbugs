## What is this Basic Javascript Directory?
* 💼 This directory serves as a simple Javascript Template.
* 🤔 It has:
	* 🕸️ Basic **HTML template**
	* 🎨 Basic **CSS templates**
		* 🖼️ **template.css** is the template css I use. It has the basic WCAG-compliance, and it is responsive to any screen size. *I suggest keeping this as your global css.*
		* **js.css** is the css that targets *that* specific button. As you build your app, you probably should rename this or delete this (all depends on your needs).
	* 💃🏻 Basic **JS template**
